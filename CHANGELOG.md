# Changelog

## 1.0.0

- Initial release of `magic_regex` for Dart.
- Core `MagicRegExp` class for readable regex construction.
- Chainable API: `.and()`, `.or()`, `.optionally()`, `.oneOrMore()`, `.zeroOrMore()`, `.times()`.
- Standard Inputs: `digit`, `word`, `whitespace`, `any`, `letter`, `exactly()`.
- Anchors: `atStart()`, `atEnd()`.
- Grouping: `grouped()`, `captured()`.
- Advanced Helpers: `anyOf()`, `charIn()`, `charNotIn()`, `maybe()`.
- Lookarounds: `before()`, `notBefore()`, `after()`, `notAfter()`.
- Full test coverage for core and complex matching scenarios.
